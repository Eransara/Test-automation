## Project Overview

This project automates the testing of the **Chat Sinhala transliteration** feature of the PixelsSuite chat translator application. The automation uses **Playwright** to input 50 chat-style Singlish test cases into the live web application and capture the actual Sinhala output. The results are saved directly into an Excel file.

All 50 test cases are **negative test cases** — scenarios where the system is expected to fail to correctly convert Singlish input into Sinhala. The test cases cover all 24 Singlish input types specified in the assignment.
